Install python-constraint from https://pypi.org/project/python-constraint/
In this project, Python-constraint CSV solver is used.
Kakuros.py is only 4 x 4 kakuro game solver.
Futoshiki.py is only 4 x 4 futoshiki game solver.
All information and constraints are provided within the program.
Input files contain dynamic value about the games.
To run python file, type "Python3 kakuros.py kakuro_input.txt" or "Python3 futoshiki.py futoshiki_input.txt" in Terminal.
